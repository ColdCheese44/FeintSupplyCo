Feint Supply Co. brand preview assets live in this directory.

- `brand-guide.json` is the source of truth for naming, tone, colors, prompts, and design rules.
- `generation-log.json` records the latest preview or generation run.
- `brand:preview` creates local reviewable PNG assets without Etsy API calls.
- `brand:setup` is reserved for the future live asset-generation flow after approval.
